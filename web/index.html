ℹ️ Voice commands & tips
Shortcuts (Alt + key):
Alt+V — Voice  |  Alt+K — Save place  |  Alt+E — Emergency

Voice commands (tap 🎙️ Voice, then speak):
• start camera / begin / start analysis — start the camera
• stop camera / end camera / shut down — stop the camera
• read text / what does it say — read text in view (silent until result)
• repeat / say again — repeat last alert aloud
• stop navigation / cancel — cancel current navigation target
• save place — open the save-location dialog
• save as <name> — save current view with a name (e.g. save as bathroom)
• emergency / help / sos / i need help — send emergency GPS
• list places / show places — list all saved places
• go to <place> / navigate to <place> — navigate to a saved place
• go home — navigate to home

About button clicks:
Pressing a button is silent — it does not announce itself. The system only speaks the result of the action (e.g. "Camera started", "Could not save place: …").

Saving a place (M4 3-second profile):
1. Start the camera first (say start camera).
2. Say save place or save as <name>.
3. Hold the camera steady at the destination for 3 seconds.
4. The system builds a 10-metric path profile automatically.

Note: Saving requires the camera to be running — the system needs live frames to build the path profile. If you say save as <name> while the camera is off, the system will refuse and tell you to start the camera first.

Arrival detection (4 reliable signals, weighted vote):
• Scene (weight 0.25) – room classifier agrees
• Object (weight 0.25) – landmark objects (e.g. toilet) detected
• Path M4 (weight 0.25) – 10-metric path profile similarity
• Visual (weight 0.15) – scene fingerprint (supporting)
Arrival is announced when the combined score is at least 0.40 and at least 2 of 4 signals agree.
